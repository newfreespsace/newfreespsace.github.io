并查集

边带权并查集​

tip

"边带权" 并查集是在普通并查集的基础上，记录每个点到其父节点的权值信息，从而支持查询“某两个节点之间的权值关系”。

在实际实现的时候，因为一般我们采用了路径压缩，所以需要处理好压缩路径的边权。

差值问题​

问题描述​

有若干条语句 a−b=wa - b = wa−b=w, 表示节点 aaa 到 bbb 的差值是 www。

给出若干这样的语句，请你判断是否有矛盾，并能查询两个变量之间的差值。

用 dist[x] 表示 x 到其根节点的差值。

查找祖先节点​

在查找祖先时使用路径压缩，并更新差值。

int get(int x) {
    if (x != fa[x]) {
        int root = get(fa[x]);
        dist[x] += dist[fa[x]];
        fa[x] = root;
    }
    return fa[x];
}
合并节点​

a−b=wa - b = wa−b=w，合并 aaa 和 bbb 所在的集合，并将 aaa 所在集合的根节点指向 bbb 所在集合的代表元素，因为 aaa 到 root_a 再到 root_b 的总长度和 aaa 到 bbb 再到 root_b 的总长度相等。所以虚线的长度应设为 w + dist[b]-dist[a]。

bool merge(int a, int b, int w) {
    int fa_a = find(a);
    int fa_b = find(b);
    if (fa_a == fa_b) {
        // 判断是否矛盾
        return dist[a] - dist[b] == w;
    }
    fa[fa_a] = fa_b;
    dist[fa_a] = dist[b] - dist[a] + w;
    return true;
}

例1​

note

有一个划分为 NNN 列的星际战场，各列依次编号为 111 , 222 ,…,NNN。

有 NNN 艘战舰，也依次编号为 111 , 222 ,…,NNN ,其中第 iii 号战舰处于第 iii 列。

有 TTT 条指令，每条指令格式为以下两种之一：

1、M i j，表示让第 iii 号战舰所在列的全部战舰保持原有顺序，接在第 jjj 号战舰所在列的尾部。

2、C i j，表示询问第 iii 号战舰与第 jjj 号战舰当前是否处于同一列中，如果在同一列中，它们之间间隔了多少艘战舰。

现在需要你编写一个程序，处理一系列的指令。

题目分析​

一列星舰构成了一个链，可视为特殊的树，因此可用并查集维护题中的关系，要检查两个战舰之间间隔的战舰，可以统计两个战舰 x 和 y 到集合根节点所需经过的战舰数量 d[x] 和 d[y]。如图，d[x]=4, d[y]=2，d[x]-d[y]-1 便是所求的答案。

![截屏2025-05-18 17.42.42](./截屏2025-05-18 17.42.42-7598703.png)

为了加快求 d[x] 的速度，我们使用路径压缩，d[x] 表示 x 与移动到其父节点所需经过的节点数量。因此，当我们执行 get 操作的时候，代码如下：

int get(int x) {
  if (X == fa[x]) return x;
  int root = get(fa[x]);
  d[x] += d[fa[x]];
  return root;
}
